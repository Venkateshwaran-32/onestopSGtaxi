# Plans

Strategic plans for OneStopSGTaxi. Each plan is a self-contained document
that names a problem, surveys the routes through it, and recommends a
specific path with concrete next steps.

## How these are different from `docs/decisions/`

- **`docs/decisions/`** — ADRs. One decision, captured at the moment it's
  made, with the alternatives considered. Short, focused, immutable once
  accepted.
- **`plans/`** — strategic horizons. Multi-phase, multi-month, with
  concrete action items and budget. Iterated as the situation changes.

If a plan converges on a single decision worth memorialising, that
decision graduates to its own ADR in `docs/decisions/`.
